## 💡 EKSANP ITILIZASYON

### Ajouterr un produit
```yaml
---
title: "T-shirt Premium"
price: 500
category: habillement
featured_image: /assets/images/tshirt.jpg
in_stock: true
variants:
  - type: Taille
    name: M
  - type: Couleur
    name: Blanc
---
Description pwodwi la...
```

### Chanje Koulè Prensipal
```css
:root {
  --primary-color: #4facfe;
  --accent-color: #00f2fe;
}
```

### Ajouter Notifikasyon
```javascript
cart.showNotification('✅ Pwodwi ajoute!', 'success');
```
